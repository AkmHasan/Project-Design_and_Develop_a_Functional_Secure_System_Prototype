Download and install XAMPP

Extract the ABCAir folder into the htdocs folder inside the XAMPP directory in your local disk
type localhost/phpmyadmin/ in your web browser to manage the database.

The database is provided so it can be imported on any system.
Click on import to select the sql file in your ABCAir folder. 
Import the sql file into the abc database
